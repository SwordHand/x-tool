# X-tool

一款专为 X 设计的油猴脚本，旨在移除敏感内容限制，并为网页端的视频提供多分辨率解析与高速下载功能。

## 🌟 主要功能

- **内容警告自动解除**：移除敏感内容限制。
- **原生下载入口**：在视频推文底部的互动工具栏（回复、转发、点赞旁边）完美嵌入原生样式下载按钮。
- **画质及码率选择**：解析并提供 1080p, 720p, 480p 等不同分辨率的视频，显示精确的比特率（Mbps）信息。
- **运行指示器**：右上角配备轻量级状态指示小绿点，随时查看脚本是否工作正常。

---

## 🚀 安装步骤

1. **安装脚本管理器**：
   在浏览器中安装 [Tampermonkey (油猴)](https://www.tampermonkey.net/) 或其它兼容的脚本管理器。

2. **添加脚本**：
   - 打开油猴管理面板，选择“添加新脚本”。
   - 将 `x-tool.js` 中的代码完全复制并粘贴进去。
   - 保存脚本。

3. **开始使用**：
   刷新或打开 [x.com](https://x.com/)，即可享受无限制浏览与便捷的一键视频下载。

---


## 🛠️ 技术声明与权限说明

为了保障解析的正常运行，脚本申请了以下特权：
- `@grant GM_xmlhttpRequest`：用于跨过浏览器 CSP 限制请求视频解析接口和获取视频数据流。
- `@grant unsafeWindow`：用于深度 Hook 网页数据清理警告。
- `@connect tweeload.aculix.net` 和 `video.twimg.com`：将数据解析和视频服务器添加至安全域名白名单，避免频繁弹框拦截。

# X-tool

一款专为 X 设计的油猴脚本，旨在移除敏感内容限制，并为网页端的视频提供多分辨率高速下载功能。

## 🌟 主要功能

- **内容警告自动解除**：移除敏感内容限制。
- **原生下载入口**：在视频推文底部的互动工具栏（回复、转发、点赞旁边）完美嵌入原生样式下载按钮。
- **画质及码率选择**：提供 1080p, 720p, 480p 等不同分辨率的视频选择，直接本地下载无需外部接口。
- **运行指示器**：右上角配备轻量级状态指示小绿点，随时查看脚本是否工作正常。

---

## 🚀 安装步骤

1. **安装脚本管理器**：
   在浏览器中安装 [Tampermonkey (油猴)](https://www.tampermonkey.net/) 或其它兼容的脚本管理器。

2. **添加脚本**：
   - 打开油猴管理面板，选择“添加新脚本”。
   - 将 `x-tool.js` 中的代码完全复制并粘贴进去。
   - 保存脚本。

3. **开始使用**：
   刷新或打开 [x.com](https://x.com/)，即可享受无限制浏览与便捷的一键视频下载。
